## How to Add Text to Images

Create stunning graphics with text overlays using our Text to Image tool.

### Step 1: Upload Your Image

1. Click "Upload Image" or drag and drop
2. Supported formats: JPG, PNG, WebP, GIF
3. Maximum file size: 10MB
4. Maximum resolution: 8000x8000px

### Step 2: Add Text

1. Click "Add Text" button
2. Type your text (supports multiple lines)
3. You can add unlimited text elements

### Step 3: Customize Your Text

**Font Selection**
- Choose from 50+ professional fonts
- Standard: Arial, Helvetica, Times New Roman
- Creative: Playfair, Bebas, Montserrat
- Decorative: Impact, Comic Sans, Brush Script

**Text Styling**
- **Font Size**: 10px to 200px
- **Font Weight**: Light, Normal, Bold, Extra Bold
- **Color**: Pick from color wheel or enter hex code
- **Background**: Add semi-transparent background
- **Opacity**: 0-100% transparency control
- **Shadow**: Add drop shadow for depth

### Step 4: Position Your Text

**Preset Positions**
- Top Left, Top Center, Top Right
- Middle Left, Center, Middle Right
- Bottom Left, Bottom Center, Bottom Right

**Manual Positioning**
1. Select text element
2. Drag on canvas to position exactly where you want
3. Resize text by dragging corners
4. Rotate text using rotation handle

### Step 5: Alignment & Spacing

- **Horizontal**: Left, Center, Right
- **Vertical**: Top, Middle, Bottom
- **Line Height**: Adjust spacing between lines
- **Letter Spacing**: Control letter-to-letter distance

### Step 6: Effects & Styling

**Text Effects**
- Outline: Add border around text
- Glow: Create luminous effect
- Skew: Distort for dynamic look
- Stroke: Outline with customizable thickness

**Advanced Options**
- Rotation: Angle from 0-360°
- Scaling: Increase/decrease size
- Blend Mode: Normal, Multiply, Screen, Overlay

### Step 7: Preview & Download

1. Preview your final image
2. Zoom in/out to check details
3. Make adjustments if needed
4. Click "Download" to save as PNG, JPG, or WebP

## Tips for Professional Results

✅ **Contrast**: Ensure text is readable against the background  
✅ **Font Size**: Make text large enough to read on small screens  
✅ **Simplicity**: Don't overcrowd with too much text  
✅ **Hierarchy**: Use size and weight to show importance  
✅ **Readability**: Avoid thin fonts on small sizes  
✅ **Spacing**: Add margins around text edges
