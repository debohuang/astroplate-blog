## Frequently Asked Questions - Text to Image

### General Questions

**Q: Is the Text to Image Tool free?**
A: Yes, completely free with unlimited usage. No registration required and no hidden fees.

**Q: Will my images be saved or shared?**
A: No, your images are only processed and never stored or shared. Complete privacy guaranteed.

**Q: Can I use the tool offline?**
A: Currently, the tool requires an internet connection. We're working on an offline version.

**Q: Is there a limit to text I can add?**
A: You can add unlimited text elements to a single image.

### Technical Questions

**Q: What image formats are supported?**
A: Input: JPG, PNG, WebP, GIF. Output: PNG, JPG, WebP (choose your preferred format).

**Q: What's the maximum image size?**
A: Maximum file size: 10MB. Maximum resolution: 8000x8000 pixels.

**Q: Can I edit text after adding it?**
A: Yes! You can edit, move, resize, and style text elements at any time before downloading.

**Q: Does the tool support Unicode characters?**
A: Yes, we support all Unicode characters including emojis, accented characters, and right-to-left languages.

**Q: Can I use custom fonts?**
A: Currently, you can choose from 50+ professional fonts. Custom font uploads coming soon.

**Q: What's the maximum text size?**
A: Text size ranges from 10px to 200px. For larger text, you can scale it proportionally.

### Design & Positioning

**Q: How do I center text on my image?**
A: Use the "Center" alignment button or manually drag text to center position on canvas.

**Q: Can I add text to only part of the image?**
A: Yes! You have full control over text positioning. Drag to place exactly where you want.

**Q: Can I make text transparent?**
A: Yes, use the Opacity slider (0-100%) to control transparency level.

**Q: How do I add text outline/stroke?**
A: Go to Text Effects → Enable Outline → Choose color and thickness.

**Q: Can I rotate text?**
A: Yes! Click the Rotation slider to rotate text 0-360 degrees.

**Q: How do I add text shadow?**
A: Go to Effects → Shadow → Adjust blur, offset, and opacity to your preference.

**Q: Can I make text curved?**
A: Currently, text follows straight paths. Curved text is in development.

### Color & Styling

**Q: How do I pick custom text color?**
A: Click the color button to open a color picker. Use the wheel, enter hex codes, or RGB values.

**Q: What are blend modes and how do I use them?**
A: Blend modes control how text interacts with the background. Try different modes for artistic effects:
- Normal: Standard overlay
- Multiply: Darkens background
- Screen: Lightens background
- Overlay: Combines multiply and screen

**Q: Can I use gradients for text color?**
A: Currently, solid colors are supported. Gradient text is coming in the next update.

**Q: How many text layers can I add?**
A: Unlimited text layers. You can create complex designs with multiple text elements.

### Fonts & Typography

**Q: What fonts are available?**
A: We offer 50+ professional fonts including:
- Sans-serif: Arial, Helvetica, Montserrat
- Serif: Georgia, Times, Playfair
- Display: Impact, Bebas, Oswald
- Script: Brush, Learning, Aladin

**Q: Can I make text bold or italic?**
A: Yes! Most fonts support multiple weights (Light, Normal, Bold, Extra Bold) and italic styles.

**Q: What's the difference between font size and font weight?**
A: Font size is the height of text (10-200px). Font weight is thickness (Light to Extra Bold).

**Q: How do I ensure text is readable?**
A: Use contrasting colors, adequate font size, and avoid complex fonts for body text. Add background for better contrast.

**Q: Can I add line breaks in text?**
A: Yes! Press Enter to create new lines. Adjust line height for spacing between lines.

### Troubleshooting

**Q: Text looks pixelated or blurry**
A: This happens with very large text sizes on low-resolution images. Try:
- Using a higher resolution image
- Reducing text size slightly
- Exporting as PNG instead of JPG

**Q: Text is hard to read on my background**
A: Tips for better readability:
- Choose contrasting color (dark text on light backgrounds)
- Add text background color
- Use text shadow for depth
- Increase font size

**Q: Download not working**
A: Try:
- Using a different browser
- Clearing browser cache
- Disabling ad blockers
- Using incognito mode

**Q: Tool is running slowly**
A: Try:
- Reducing image resolution slightly
- Closing other applications
- Refreshing the page
- Using a different browser

**Q: Text position changed after download**
A: This can happen with different aspect ratios. Make sure to:
- Preview before downloading
- Check final size matches preview
- Position text with margins from edges
