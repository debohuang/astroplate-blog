## Frequently Asked Questions

### General Questions

**Q: Is the Image Merge Tool free?**
A: Yes! The tool is completely free and unlimited. No registration or credit card required.

**Q: Does it work on mobile?**
A: Yes, the tool is fully responsive and works on smartphones, tablets, and desktops.

**Q: Is my data secure?**
A: All images are processed locally in your browser or on our secure servers. Images are automatically deleted after processing.

### Technical Questions

**Q: What file formats are supported?**
A: We support JPG, PNG, WebP, GIF, and BMP formats for input. Output is available as PNG, JPG, or WebP.

**Q: What's the maximum file size?**
A: Each image can be up to 10MB, and the combined result up to 50MB.

**Q: What's the maximum resolution?**
A: The maximum resolution is 8000x8000 pixels. For larger images, we'll automatically optimize.

**Q: Can I merge more than 9 images?**
A: Yes! Simply upload as many images as you need. The tool supports unlimited image merging.

**Q: How is the final image resolution calculated?**
A: The resolution depends on your merge format and original image sizes. For horizontal merge, width is sum of all widths, height is maximum height.

### Usage Questions

**Q: How do I arrange images in a specific order?**
A: After uploading, you can drag and drop images to reorder them before merging.

**Q: Can I adjust individual image sizes?**
A: Yes, you can resize images before merging. Custom dimensions can be set for each image.

**Q: What happens if images have different sizes?**
A: The tool automatically handles different sizes. You can choose to:
- Scale all images to match
- Fill empty space with a background color
- Maintain original sizes with padding

**Q: Can I add spacing between images?**
A: Yes! Choose from predefined spacing options or set custom pixel values (1-50px).

**Q: How long does merging take?**
A: Most merges complete within 1-3 seconds, depending on image sizes and your internet speed.

### Troubleshooting

**Q: The tool is slow or unresponsive**
A: Try:
- Refreshing the page
- Using a different browser
- Reducing image file sizes
- Closing other applications

**Q: Images appear blurry after merging**
A: This might happen with very large images. Try:
- Using the "High Quality" output setting
- Using PNG format instead of JPG
- Reducing image dimensions slightly

**Q: Download not working**
A: Try:
- Using a different browser
- Clearing browser cache
- Disabling browser extensions
- Using incognito/private mode

**Q: Getting "Processing Error"**
A: This usually means:
- One or more images are corrupted
- Try uploading different images
- Use simplified image formats (JPEG)

### Privacy & Security

**Q: Who has access to my images?**
A: Only you. Images are not stored or shared with anyone.

**Q: Where is my data processed?**
A: Images are processed either in your browser or on secure servers with automatic deletion after processing.

**Q: Can I trust this tool?**
A: Yes, our privacy policy is transparent and we comply with GDPR and data protection regulations.

### Advanced Questions

**Q: Can I use this for batch processing?**
A: Yes! Our batch processing feature lets you merge multiple sets of images automatically.

**Q: How do I use the API for programmatic merging?**
A: We offer a REST API for developers. Contact us for API documentation and keys.

**Q: Can I schedule automated image merging?**
A: For business users, we offer scheduled batch processing. Contact our sales team.
